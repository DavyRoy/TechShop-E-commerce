About TechShop

Our Story

TechShop was founded in 2026 as a DevOps learning project. This e-commerce platform serves as a practical example for mastering containerization, CI/CD, infrastructure as code, and cloud technologies.

Our Mission

To provide hands-on experience with modern DevOps practices while building a realistic e-commerce application from the ground up.

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Web Server: Nginx
  • Containerization: Docker
  • Version Control: Git & GitHub
  • CI/CD: GitHub Actions (coming soon)
  • Orchestration: Kubernetes (coming soon)
